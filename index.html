Conoce StressMeter

En el ajetreo diario, es fácil que el estrés se acumule sin que te des cuenta. Pero, ¿y si pudieras ponerle un número? Bienvenido a StressMeter, el punto de partida para recuperar el control.

Nuestro Sistema Experto

StressMeter opera con un sofisticado Sistema Experto. Este sistema es un programa diseñado para emular el razonamiento de un experto en la materia. Su funcionamiento se basa en un robusto algoritmo de validación que procesa las respuestas del usuario utilizando como base la Escala de Depresión (DASS-21).

La DASS-21 es una herramienta psicométrica reconocida a nivel internacional, utilizada para evaluar la severidad de los síntomas emocionales. Nuestro sistema experto se enfoca en los ítems validados de la escala de estrés para analizar respuestas de manera precisa, validar la consistencia de los síntomas reportados, y cuantificar el nivel de estrés, clasificándolo de forma objetiva (por ejemplo: normal, leve, moderado o severo).
